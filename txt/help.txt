help       - print a list of available commands
echo       - print a value, it will be interpreted so use quotes if youre printing a string
install    - install packages from the cideros github, use as `install txt/help.txt`, make sure to include path
cdrl       - run cdrl programs, optionally add args, these args will be raw, not interpreted. use as `cdrl test.cdrl "arg1", "arg2"`
